# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please send an email to [mo.mn@gmail.com] with **Security Issue** in the subject line. Please include:

- A clear description of the vulnerability
- Reproduction steps or proof-of-concept code
- Impact assessment and any mitigation strategies

You can expect a response within 7 business days.

## Supported Versions
| Version | Supported          | Security Fixes        |
| ------- | ------------------ | --------------------- |
| `main`  | :white_check_mark: | :rocket: Continuous   |

## Acknowledgements
Thank you to all security researchers who report issues responsibly.
